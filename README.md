# Программа получает информацию о вакансиях с платформы hh.ru в России.



***Пользователь*** задает параметры поиска ванасий:
- название вакансии
- количесво вокансий
- миниальную заработную плату
- город

Получает ответ по заданным параметрам:
- Город
- Дата публикации
- Должность
- Требование
- Ответственность
- Зарплата
